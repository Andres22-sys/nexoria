# Nexoria

Nexoria is a modern, responsive static website built using [Vite](https://vitejs.dev/) and React. It combines clean design and cutting-edge technology to deliver a fast, seamless user experience. Explore its features, elegant design, and mobile-friendly layouts.

## Features
- **Responsive Design**: Optimized for both mobile and desktop devices.
- **Modern Tech Stack**: Built with React and Vite for fast builds and development.
- **Performance-First Approach**: Lightweight, fast-loading, and secure.
- **Clean UI**: Elegant interface with intuitive navigation.
- **Scalable Architecture**: Easily extendable and customizable.

## Tech Stack
- **React**: Frontend framework for UI components.
- **Vite**: Blazing-fast development build tool.
- **CSS**: Responsive and modern styles.
- **FontAwesome**: For beautiful, scalable icons.

## Installation

Follow these steps to set up Nexoria locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/<your-username>/nexoria.git
   cd nexoria
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser and go to:
   ```
   http://localhost:5173
   ```

## Build for Production

To create a production-ready build, run:
```bash
npm run build
```
The build will be output to the `dist` folder.

## Deployment

### On Vercel (Recommended)
1. Build the project:
   ```bash
   npm run build
   ```
2. Deploy the `dist` folder to [Vercel](https://vercel.com/).

### Other Hosting Platforms:
- **Netlify**: Drag and drop the `dist` folder to Netlify.
- **GitHub Pages**: Use `gh-pages` for deployment.
- **Firebase Hosting**: Deploy with Firebase CLI.
